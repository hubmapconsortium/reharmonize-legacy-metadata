3. Conditional Patches

Rule-based transformations applied when specific conditions are met.

  • #1 When the LC instrument model is 'Ultimate 3000 RSLCnano system': Sets LC instrument model as 'UltiMate 3000 RSLCnano UHPLC System', lc instrument vendor as 'Thermo Fisher Scientific'.
  • #2 When the protocol DOI is '10.17504/protocols.io.bf6bjran' and the acquisition instrument model is 'Orbitrap Lumos':
    • Set LC instrument model to 'Waters; ACQUITY UPLC M-Class System with 2D Technology'
    • Set lc instrument vendor to 'Waters'
    • Set lc column model to 'Waters Xbridge C18 column (300m x 5 cm, 5.0m);In-House'
    • Set lc column vendor to 'Waters'
    • Set lc resin to 'BEH|HPLC Column;Sepex GP-C18 1.8um, 120A'
  • #3 When the protocol DOI is '10.17504/protocols.io.bf4ujqww' and the acquisition instrument model is 'Q Exactive': Sets lc column model as 'Agilent Technologies; Zorbax SBaq column 2.1 x 50 mm, 1.7 m, 100A'.
  • #4 When the protocol DOI is '10.17504/protocols.io.bf4ujqww' and the acquisition instrument model is 'Q Exactive HF': Sets lc column model as 'Millipore Sigma; SeQuant ZIC-HILIC 200A (3.5 m) HPLC Columns; PN 1.50447', lc column vendor as 'Millipore'.
  • #5 When the protocol DOI is '10.17504/protocols.io.bf4vjqw6' and the LC instrument model is 'LC-30AD': Sets data collection mode as 'DIA', ms scan mode as 'MS2'.
  • #6 When the protocol DOI is '10.17504/protocols.io.bft3jnqn': Sets ms scan mode as 'MS2', lc column vendor as 'In-House'.
  • #7 When the protocol DOI is '10.17504/protocols.io.bf6bjran' and the acquisition instrument model is 'Orbitrap Fusion Tribrid': Sets acquisition instrument model as 'Orbitrap Fusion Lumos Tribrid', assay type as 'LC-MS'.
  • #8 Sets ms scan mode as 'MS2'.
