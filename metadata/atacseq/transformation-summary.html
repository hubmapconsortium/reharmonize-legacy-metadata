3. Conditional Patches

Rule-based transformations applied when specific conditions are met.

  • #1 When the assay type is 'snATACseq':
    • Set input entity to 'single nucleus'
    • Set barcode read to 'Read 2 (R2)'
    • Set barcode size to '16'
    • Set barcode offset to '0'
    • Set UMI read to 'Not applicable'
    • Set UMI size to 'Not applicable'
    • Set UMI offset to 'Not applicable'
  • #2 When the assay type is 'bulkATACseq':
    • Set input entity to 'tissue (bulk)'
    • Set barcode read to 'Not applicable'

    • Set UMI read to 'Not applicable'

  • #3 When the assay type is 'sciATACseq':
    • Set input entity to 'single cell'
    • Set barcode read to 'Not applicable'
    • Set barcode size to 'Not applicable'
    • Set barcode offset to 'Not applicable'
    • Set UMI read to 'Not applicable'
    • Set UMI size to 'Not applicable'
    • Set UMI offset to 'Not applicable'
  • #4 When the assay type is 'SNARE-seq2':
    • Set dataset type to 'ATACseq'
    • Set input entity to 'single nucleus'
    • Set barcode read to 'Read 2 (R2)'
